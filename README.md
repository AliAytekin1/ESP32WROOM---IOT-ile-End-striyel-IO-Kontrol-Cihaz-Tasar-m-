# 🌐 ESP32 MQTT IoT Dashboard

Bu proje, **ESP32** tabanlı IoT cihazlarının **MQTT protokolü** ile haberleştiği, **Node.js** sunucusu üzerinden verilerin işlendiği ve **web tabanlı bir arayüzle gerçek zamanlı izlenebildiği** bir kontrol paneli sistemidir.

## 📌 Özellikler

- ✅ ESP32 cihazlarıyla çift yönlü MQTT iletişimi  
- ✅ WebSocket ile gerçek zamanlı veri güncelleme  
- ✅ Kullanıcı dostu, responsive web arayüzü (HTML, CSS, JS)  
- ✅ LED kontrolü ve komut gönderme özelliği  
- ✅ MongoDB üzerinde veri kaydı ve sorgulama  
- ✅ Dinamik grafikler (Chart.js ile)  
- ✅ Cihaz başlığı güncelleme ve çevrimiçi durum takibi

---

## 🧱 Kullanılan Teknolojiler

- ESP32 (Arduino/C++)
- MQTT (Mosquitto Broker)
- Node.js + Express.js
- MongoDB + Mongoose
- WebSocket (Socket.IO)
- HTML5 / CSS3 / JavaScript
- Chart.js
